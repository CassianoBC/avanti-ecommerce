# 🛍️ E-commerce Avanti

![image](https://raw.githubusercontent.com/CassianoBC/avanti-ecommerce/refs/heads/main/src/assets/images/readme-img.png)

Interface de um e-commerce moderno desenvolvida com HTML, CSS, JavaScript puro e TailwindCSS. O projeto foca em interatividade e responsividade.

## 🚀 Tecnologias utilizadas

HTML5

CSS3

JavaScript (Vanilla)

TailwindCSS

Swiper.js (slider)


## ✨ Funcionalidades

🧭 Menu com dropdown interativo

🖼️ Slider/carrossel de produtos

📱 Layout totalmente responsivo


## 🛠️ Como executar o projeto
1. **Clone o repositório:**
   
   ```bash
   https://github.com/CassianoBC/avanti-ecommerce.git
   ```
   
2. **Entre na pasta**

    ```bash
   cd avanti-ecommerce
   ```

3. **Instale as dependências**

    ```bash
   npm install
   ```
  
4. **Abra o arquivo index.html no navegador**

     Recomendo uso da extensão Live Server
